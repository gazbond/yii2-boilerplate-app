FULL NPM/NODE/GULP UPDATE
-------------------------

    sudo npm cache clean -f
    sudo npm install npm -g
    sudo npm install -g n
    sudo n stable
    sudo npm install gulp -g
    sudo gem install scss_lint


INSTALL LOCAL NPM AND RUN GULP
------------------------------

    npm install
    gulp

    # watch for ts/scss etc. changes
    gulp watch
